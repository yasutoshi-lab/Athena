# ログイン画面

## 概要

Athena にアクセスする際の認証画面です。ユーザー名とパスワードを入力してログインします。

## 画面配置

```
┌─────────────────────────────────────────────┐
│                                             │
│           ┌─────────────────────┐           │
│           │  [A] Athena          │           │
│           │                     │           │
│           │  ログイン            │           │
│           │  因果推論システムに... │           │
│           │                     │           │
│           │  ユーザー名          │           │
│           │  ┌─────────────────┐│           │
│           │  │ admin           ││           │
│           │  └─────────────────┘│           │
│           │                     │           │
│           │  パスワード          │           │
│           │  ┌─────────────────┐│           │
│           │  │ ••••••••        ││           │
│           │  └─────────────────┘│           │
│           │                     │           │
│           │  [  ログイン  ]     │           │
│           │                     │           │
│           │  Athena v0.1        │           │
│           └─────────────────────┘           │
│                                             │
└─────────────────────────────────────────────┘
```

## 使い方

1. **ユーザー名**を入力します
2. **パスワード**を入力します
3. **「ログイン」ボタン**をクリック、または **Enter キー**で送信します
4. 認証成功後、自動的にメイン画面に遷移します

## 補足

- ログイン失敗時は「ログインに失敗しました」とエラーが表示されます
- JWT（JSON Web Token）方式で認証しています
- アクセストークンは 1 時間有効、リフレッシュトークンは 7 日間有効です
- トークンが残っている場合、次回アクセス時はログイン画面をスキップして直接メイン画面に遷移します
- 多言語対応済み（設定画面で日本語/English を切り替え可能）
