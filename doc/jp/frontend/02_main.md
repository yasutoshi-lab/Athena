# メイン画面（推論チャット + 知識グラフ）

## 概要

Athena のメイン画面です。3 パネル構成で、左からセッション一覧・推論チャット・知識グラフが配置されます。因果的な問いを入力すると、AI が 7 段階のパイプラインで分析し、結果をリアルタイムに表示します。

│            │                         │                           │

## 3 パネル構成

### 1. セッション一覧サイドバー（左・220px）

- **＋ 新規セッション**: 新しいセッションを開始
- **セッション一覧**: 日付でグループ分け（今日 / 昨日 / それ以前）
- **ステータスドット**: 緑=実行中、青=完了、赤=エラー
- **三点リーダーメニュー(···)**: ホバーで表示、「編集」「削除」操作
- **✕ ボタン**: サイドバーを閉じる（TopBar の「セッション一覧」で再表示）

### 2. 推論チャットパネル（中央・380px、リサイズ可能）

推論の進行状況と結果がリアルタイムで表示されます。

**表示されるメッセージの種類:**

| 種類 | 説明 |
|------|------|
| ユーザーメッセージ | 入力した問い（右寄せ・青背景） |
| ステップ通知 | パイプライン各ノードの完了通知 |
| モデル決定 | 使用モデル・複雑度スコア・エンティティ数 |
| 構造化クエリ | 主語・述語・スコープ・時間軸・エンティティ |
| 仮説カード | 生成された因果仮説（スコア付き） |
| 検索アクティビティ | Web 検索の進行状況 |
| 証拠サマリ | 証拠収集の合計・支持数・反証数 |
| 最終回答 | 参照元リンク付きの分析結果（コピーボタン付き） |

**操作:**
- **Enter**: メッセージ送信
- **Shift + Enter**: 改行
- **停止ボタン**: 実行中のパイプラインを中断
- **クリアボタン**: チャットとグラフをリセット

### 3. 知識グラフパネル（右・flex:1）

D3.js によるインタラクティブなフォース有向グラフ。

**ノードの種類:**

| 色 | 種類 | 説明 |
|----|------|------|
| 紫 | Question | 入力された問い |
| 黄 | Hypothesis | 生成された仮説 |
| 緑 | Support | 支持する証拠 |
| 赤 | Counter | 反証する証拠 |
| 灰 | Concept | 概念ノード |

**操作:**
- **ドラッグ**: ノードを移動
- **スクロール**: ズームイン/アウト
- **クリック**: ノードの詳細情報を右下に表示
- **ホバー**: 接続されたノードをハイライト
- **フィルタボタン**: ノード種別の表示/非表示切り替え

## パネル間のリサイズ

チャットパネルと知識グラフの境界にあるドラッグハンドルをドラッグすることで、パネル幅を調整できます。

## フォローアップ質問

初回の推論完了後、同じセッション内で追加の質問が可能です。パイプラインは再実行されず、既存の分析結果をコンテキストとして Claude が直接回答します。

- 例: 「仮説 1 についてもっと詳しく教えてください」
- 知識グラフや仮説は保持されたまま、チャットに回答が追加されます
