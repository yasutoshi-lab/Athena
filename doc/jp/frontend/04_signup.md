# アカウント作成画面

## 概要

新規ユーザーが自分でアカウントを作成するための画面です。ユーザー名・メールアドレス・パスワードを入力して登録します。登録完了後は自動的にログインし、メイン画面に遷移します。

## 使い方

1. ログイン画面で **「新規登録」リンク** をクリックして遷移します
2. **ユーザー名** を入力します（他のユーザーと重複不可）
3. **メールアドレス** を入力します（他のユーザーと重複不可）
4. **パスワード** を入力します
5. **パスワード（確認）** に同じパスワードを再入力します
6. **「アカウントを作成」ボタン** をクリックします
7. 登録成功後、自動的にログインしてメイン画面に遷移します

## 入力項目

| 項目 | 必須 | 説明 |
|------|------|------|
| ユーザー名 | はい | 最大150文字。ログイン時に使用します |
| メールアドレス | はい | 有効なメールアドレス形式 |
| パスワード | はい | Djangoのパスワードポリシーに準拠（8文字以上、数字のみ不可、一般的すぎるパスワード不可） |
| パスワード（確認） | はい | パスワードと一致する必要があります |

## バリデーション

以下の場合にエラーメッセージが表示されます：

- ユーザー名が既に使用されている場合
- メールアドレスが既に登録されている場合
- パスワードが弱い場合（短すぎる、数字のみ、一般的すぎる等）
- パスワードとパスワード（確認）が一致しない場合
- 必須フィールドが未入力の場合

## 言語切り替え

画面右上の **JP / EN** ボタンで表示言語を日本語と英語に切り替えられます。切り替えはログイン画面・アカウント作成画面で共通して利用できます。

## 補足

- 外部認証（Google、GitHub 等）には対応していません。ユーザー名とパスワードによる認証のみです
- 登録時に UserSettings（ユーザー設定）が自動作成されます
- 登録成功と同時に JWT トークンが発行され、追加のログイン操作は不要です
- 「ログイン」リンクからログイン画面に戻ることができます

## API エンドポイント

| エンドポイント | メソッド | 説明 |
|---|---|---|
| `/api/auth/register/` | POST | 新規ユーザー登録。成功時にJWTトークンを返却 |

### リクエスト例

```json
{
  "username": "newuser",
  "email": "newuser@example.com",
  "password": "SecurePass123!",
  "password_confirm": "SecurePass123!"
}
```
