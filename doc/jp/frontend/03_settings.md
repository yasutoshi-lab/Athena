# 設定画面

## 概要

ユーザープロフィール、LLM モデル設定、API 使用量の確認、外観カスタマイズを行う画面です。

## 画面配置

```
┌──────────────────────────────────────────────────────────────────┐
│ [A] Athena │ 推論チャット  設定                 │ Sonnet — 待機中 │
├────────────┬─────────────────────────────────────────────────────┤
│            │                                                     │
│  設定       │  一般                                               │
│            │  プロフィールとモデルの設定                           │
│  ⊞ 一般    │                                                     │
│  ◷ API使用量│  プロフィール                                       │
│  ◑ 外観    │  ┌───────────────────────────────────────────┐      │
│            │  │ 氏名        [________________]            │      │
│            │  │ 呼び名      [________________]            │      │
│            │  └───────────────────────────────────────────┘      │
│            │                                                     │
│            │  言語                                               │
│            │  ┌───────────────────────────────────────────┐      │
│            │  │ 表示言語    [日本語 ▼]                    │      │
│            │  └───────────────────────────────────────────┘      │
│            │                                                     │
│            │  LLM モデル                                         │
│            │  ┌───────────────────────────────────────────┐      │
│            │  │ デフォルトモデル  [自動判定（推奨） ▼]     │      │
│            │  │ 複雑度しきい値    [3]                     │      │
│            │  └───────────────────────────────────────────┘      │
│            │                                                     │
│            │  [変更を保存]                                       │
│            │                                                     │
│  → ログアウト│                                                    │
└────────────┴─────────────────────────────────────────────────────┘
```

## タブ一覧

### 一般

| 項目 | 説明 |
|------|------|
| 氏名 | システム内での表示名 |
| 呼び名 | Athena がチャットで使う名前 |
| 表示言語 | 日本語 / English の切り替え（即座に反映） |
| デフォルトモデル | 自動判定 / Claude Sonnet / Claude Opus |
| 複雑度しきい値 | Opus に切り替えるエンティティ数（デフォルト: 3） |
| システムプロンプト | 全セッション共通のカスタムプロンプト |

### API 使用量

| 項目 | 説明 |
|------|------|
| 総コスト | 今月の合計コスト（USD） |
| 総トークン | 入力/出力トークン数 |
| セッション数 | Sonnet / Opus 別の実行回数 |
| モデル別コスト | Sonnet / Opus のコスト比率バー |
| セッション一覧 | 直近 20 セッションのトークン・コスト詳細 |

### 外観

| 項目 | 説明 |
|------|------|
| カラーモード | ダーク / ライト / システム |
| ノードアニメーション | 推論進行時のフェードイン ON/OFF |
| グリッド表示 | グラフ背景のグリッド ON/OFF |
| アニメーション速度 | 低速 / 標準 / 高速 |

## 使い方

1. 左メニューからタブを選択します
2. 設定を変更します
3. **「変更を保存」ボタン**をクリックして保存します
4. 言語設定は変更と同時に即座に反映されます（保存も必要）

## ログアウト

左メニュー最下部の「→ ログアウト」をクリックするとログアウトし、ログイン画面に戻ります。
